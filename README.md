# avshell

Simple shell for an OS Course

## Build guide 

### Prerequisites

- **CMake** 
- **C Compiler**
- **MSYS2 UCRT64** 

### Build steps

1. **Generate build files with CMake:**
   ```bash
   cmake -B build
   ```

3. **Compile the project:**
   ```bash
   cmake --build build
   ```

4. **Run the shell:**
   ```bash
   ./build/avshell.exe
   ```
